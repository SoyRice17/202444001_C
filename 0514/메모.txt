
자료형 배열이름 [*] *부분에 변수는 못들어감, (실제로는 사용가능하지만 시험에서는 안되는거로 작성)

int a = 10; a = 값

int arr[100]; arr = 배열의 시작 주소,arr[0]

문자열 저장 
scanf("%s", a) a[0] 부터 차레 저장
scanf("c", &a[0]) 특정지점 저장 원하면 %c, &변수 형식으로 작성

Sizeof(arr) 해당함수는 바이트 크기를 반환해줌 arr으로 입력하면 400이 반환될것임;

배열 크기가 5라고 하고 5개의 값을 넣으면 5번 배열에는 0이 초기화됨

#include <stdio.h>

int main () {
    int arr[5] = {1,2,3,4,5};

    for(int i = 0; i < 6; i++){
        printf("arr[%d] = %d\n",i,arr[i]);
    }
}
